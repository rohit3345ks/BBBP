SpriteFont+ is a Construct 2 plugin that extends the existing Sprite Font native plugin to include richer support for individual character widths during editing and runtime. You can use it in the free edition as well as all paid editions. The license is totally open so feel free to use it in your commercial or personal projects, etc. No attribution necessary.

 

Find out all about how to install and use the plugin over on the Documentation tab.
